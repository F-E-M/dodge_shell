# dodge_shell
 A simple game made with pygame
 
Language:  Chinese

how to run:
1. download python (if you already download it, skip this step)
2. download files
3. run main.py

how to play:
  use keyborad:
    [left] move left
    [right] move right
    [space] Electromagnetic wave
    [P] pause game
  use fingertip piano:
    [joystick] move
    [button A] Electromagnetic wave
    [button B] pause game
